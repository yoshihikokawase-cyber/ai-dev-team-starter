# 100日チャレンジ Skills — インデックス

このフォルダーは `.claude/commands/` に配置したスキルの参照用インデックスです。
スラッシュコマンドの実体は `.claude/commands/` にあります。

## スキル一覧

| コマンド | ファイル | 用途 |
|---|---|---|
| `/pm` | `.claude/commands/pm.md` | 今日の計画・MVPスコープ定義 |
| `/ba` | `.claude/commands/ba.md` | ミニ仕様書・受け入れ条件作成 |
| `/eng` | `.claude/commands/eng.md` | 最小差分実装 |
| `/test` | `.claude/commands/test.md` | Playwright + Antigravity テスト |
| `/xpost` | `.claude/commands/xpost.md` | X投稿文（150〜220文字）作成 |
| `/day-run` | `.claude/commands/day-run.md` | 全工程一括実行 |

## 使い方

```
/pm ToDoアプリのタグ絞り込み機能
/ba ユーザーがタグでTodoを絞り込める機能
/eng Todoタグ絞り込みフィルタを実装する
/test Todo絞り込み機能（http://localhost:3000）
/xpost Day34: タグ絞り込み機能を実装。フィルタのUXで詰まったがスッキリ解決した
/day-run 習慣トラッカーのストリーク表示機能
```
